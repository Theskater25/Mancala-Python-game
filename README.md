# Mancala-Python-game
Jeu de Mancala programmé en Python avec Pygame. Deux joueurs déplacent des billes sur un plateau de 12 trous et deux trésors. Les billes sont distribuées une par une, avec possibilité de rejouer ou de capturer celles de l’adversaire. La partie se termine lorsqu’un côté est vide. Le joueur avec le plus de billes dans son trésor gagne.
