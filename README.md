Mancala (Python / Pygame)
Ce projet est une adaptation du jeu de société Mancala, réalisée en Python avec la bibliothèque Pygame. Le jeu se joue à deux joueurs sur un plateau composé de 12 trous et de deux trésors. Le but est de récupérer plus de billes que son adversaire en appliquant les règles de distribution et de capture.

Règles du jeu :
  -Chaque trou contient 4 billes au début de la partie.
  -Les joueurs jouent chacun leur tour.
  -Un joueur choisit un trou de son côté et distribue les billes une par une dans les trous suivants.
  -Si la dernière bille tombe dans son trésor, le joueur rejoue.
  -Si la dernière bille tombe dans un trou vide de son côté, il capture les billes du trou opposé.
  -La partie se termine lorsqu’un côté du plateau est vide.
  -Le joueur avec le plus de billes dans son trésor gagne.

Contrôles :
  -Clic gauche : sélectionner un trou
  -Touche R : recommencer la partie
  -Fermer la fenêtre : quitter le jeu
